1) What is the difference between null and undefined?
ANS : Null মনে empty value, undefined মনে value declear করা আছে কিন্তু value দেয়া নাই

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
ANS : map() নতুন array return করে, কিন্তু foreach() কিছুই return করে না

3) What is the difference between == and ===?
ANS : == দিয়ে value compare করে, === দিয়ে একই সাথে value + data type compare করে

4) What is the significance of async/await in fetching API data?
ANS : Asynchronous code কে synchronous মতো readable করে , .then() chain avoid করা যায়
